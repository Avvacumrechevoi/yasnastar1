# Визуальная проверка экрана «Звезда» от 2026-04-12

После переработки экран собран в три логические рабочие зоны: слева фильтры и механики, по центру сцена звезды, справа карточка-конструктор. На текущем viewport все три области видны одновременно без вертикального скролла страницы.

| Зона | Наблюдение | Вывод |
| --- | --- | --- |
| Левая колонка | Группы «Ясны», «Переговоры и проблемы» и «Механики звезды» читаются отдельно и не налезают друг на друга | Компоновка стала логичнее и компактнее |
| Центральная сцена | Звезда остаётся главным акцентом, подписи полочек не выходят за пределы сцены, ось 0/6 читается снизу и сверху | Основная рабочая зона удержана без обрезаний |
| Правая колонка | Карточка-конструктор держит шаблон табы → описание → механика → активная полочка | Пользовательский сценарий стал последовательнее |
| Общий экран | Блоки помещаются в один экран, критичных наложений не видно | Можно переходить к финальной проверке и сохранению checkpoint |

Отдельно отмечено, что выбор табов теперь ограничен двумя одновременно. Это уменьшает перегруз правой карточки и поддерживает формат объединения через слеш.

## Повторная интерактивная проверка

После переключения одной из вкладок комбинация изменилась на «Переговоры / Атмосфера». Правая карточка сразу обновила название, описание и формулировку активной полочки без сдвигов, переполнения и поломки сетки.

| Проверка | Результат |
| --- | --- |
| Ограничение до двух табов | Работает: отображается только актуальная пара |
| Центральный заголовок сцены | Обновляется синхронно с выбранной парой |
| Правая карточка-конструктор | Сохраняет последовательную структуру и остаётся читаемой |
| Общая компоновка | На одном экране без лишнего вертикального скролла |
