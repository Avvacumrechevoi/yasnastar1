# Визуальная проверка обновлённой страницы звезды

## Что подтверждено

| Проверка | Результат |
|---|---|
| Кнопка «Выбрать все» | Работает: счётчик меняется на 14/14. |
| Кнопка «Снять все» | Доступна после массового выбора. |
| Новый список механик | В панели отображаются 14 механик: 3 креста, 4 праны, 4 оси / линии и 3 дуги. |
| Центральная звезда | После массового выбора одновременно показываются полигоны, линии и дуги. |
| Правая панель «Активно» | Отображает полный список выбранных механик в виде чипов. |

## Наблюдение
Интерфейс остался чистым и читаемым, но при выборе всех механик центральная звезда становится очень насыщенной по линиям. Это ожидаемое поведение для режима полного наложения всех структур.
