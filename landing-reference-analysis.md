# Анализ референса для нового главного экрана Ясны

## Что именно работает в референсе

| Элемент | Как устроено | Что переносим в Ясну |
|---|---|---|
| Первый экран | В центре находится одно ценностное обещание, без конкурирующих панелей и вторичных сценариев | Первый экран Ясны должен стать разводящим: один главный оффер про освоение Ясны и успешные переговоры |
| Хедер | Логотип слева, короткая навигация справа, минимум уровней и минимум визуального шума | В верхней навигации Ясны нужны только основные входы: метод, инструменты, кейсы, знания, вход в работу |
| Главный CTA | Одна яркая кнопка в центре экрана, которая собирает внимание и запускает основной сценарий | У Ясны нужен один основной сценарий: «Изучить Ясну» или «Начать разбор переговоров» |
| Средний пояс преимуществ | Под оффером идут 3 короткие опоры, которые быстро объясняют ценность продукта | Под оффером Ясны стоит показать 3 причины, зачем осваивать метод: яснее видеть конфликт, быстрее собирать структуру разговора, переходить к решению |
| Нижняя полоса инструментов | Инструменты поданы как самостоятельные кликабельные сущности с иконкой и подписью | Методы Ясны должны выглядеть как отдельные интерактивные входы в разделы: атмосфера, облака, образы, решения, синтез, кейсы |
| Визуальный ритм | Сильный центр, затем короткие пояснения, затем кликабельные модули | Новая страница Ясны должна вести пользователя сверху вниз: оффер → преимущества → инструменты |

## Принципы, которые особенно важны для Ясны

| Принцип | Практический вывод для экрана |
|---|---|
| Один смысловой центр | Убираем ощущение дашборда с первого экрана; вместо этого делаем лендинг-хаб |
| Ясная иерархия | Сначала обещание результата, потом пояснение, потом выбор инструмента |
| Кликабельные методы | Каждый инструмент Ясны должен быть отдельной карточкой с мягкой анимацией и явным переходом |
| Мягкая демонстрация глубины | Не показывать все рабочие данные сразу; глубина должна открываться после выбора раздела |
| Спокойный премиальный ритм | Минимум визуального шума, крупная типографика, ограниченная палитра, точечные акценты |

## Предлагаемая структура нового экрана Ясны

1. Верхний хедер с логотипом и короткими разделами.
2. Центральный hero-блок с оффером в духе «Изучи Ясну и веди переговоры осмысленнее».
3. Один главный CTA и один вторичный CTA.
4. Короткий ряд из трёх преимуществ метода.
5. Большая секция инструментов и методов Ясны в виде кликабельных анимированных карточек.
6. Переходы из карточек в соответствующие разделы или якоря страницы.

## Дизайн-намерение

Новый первый экран Ясны должен восприниматься не как рабочая панель фасилитатора, а как вход в метод: спокойный, уверенный, современный, с ощущением, что пользователь попадает в систему, которая помогает лучше думать о переговорах, а не просто показывает много блоков одновременно.
