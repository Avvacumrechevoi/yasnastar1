# Исследование: Атмосфера и Облака в Ясне

## Из Рубрикатора Полочек
**Ясна Круговорота Воды:**
- 0: Вода с землей-грязь
- 1: Болото / Ключ / Родник
- 2: Река
- 3: Поверхность водоёма
- 4: Пар
- 5: Облако
- 6: Холод / переход-перенос
- 7: (пропущена в таблице — Молния/Гром/Туча)
- 8: Дождь
- 9: Касание воды
- 10: Стекание
- 11: Лужа (брызгает-стреляет)

## Из Заморского "Атмосферные явления" (1954)
**6 групп атмосферных явлений:**
1. Отложения воды и льда на поверхности (роса, иней, наморозь, гололёд)
2. Осадки из воды и льда (морось, дождь обложной, дождь ливневой, снег, град)
3. Помутнение воздуха (туман, дымка, ледяные кристаллы, метель, пыль)
4. Сильные ветры
5. Оптические явления
6. Электрические явления

**Классификация по классам и видам:**
- Каждая группа → несколько классов → несколько видов
- Пример: туман = 5 видов (радиационный, адвективный, испарения, сибирский, адиабатический)

## Из Ясна Облачности (Урок 6)
- Облачность = 5-я полочка (Облако)
- Связана с цветами радуги
- Нужно прочитать подробнее

## Из Урока 5 Круговорота Воды
- 3: поверхность водоёма → волны, буруны, барашки
- 4: испарение, пар, конденсация → в облака (сверху холод)
- 5: облака → ветер носит по небу
- 6: переход → образование тучи
- 7: туча накапливается → гром и молния, лев
- 8: дождь → дева слёзы, мокрая собака
- 9: касание → вода коснулась земли
- 10: стекание → промокает земля
- 11: лужи → брызги, солнечные зайчики

## Задание пользователя
1. 4 ясны в описании переговоров
2. Атмосфера по ясне
3. Облака = проблемы
4. Образ проблем: разложить облака (перистые, кучевые и т.д.)
5. Против каждого облака подобрать вид проблем
6. Способы решения: управление ветром/воздухом → управленческие приёмы
